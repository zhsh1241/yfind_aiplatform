# yfind_aiplatform Frontend

React + TypeScript management console baseline for `F001-platform-architecture-baseline`.

## Commands

```powershell
Push-Location frontend; npm install; Pop-Location
Push-Location frontend; npm run lint; Pop-Location
Push-Location frontend; npm run test:ci; Pop-Location
Push-Location frontend; npm run build; Pop-Location
Push-Location frontend; npm run e2e; Pop-Location
```

This baseline intentionally includes only the platform shell. Business pages start in later feature packages.
