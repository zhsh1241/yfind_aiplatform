2026-05-07

## 实现范围

- 后端提供数据集列表、详情、申请与审批 API；后端不可用时回退到本地 fallback。
- 前端数据资产工作台支持列表、搜索、筛选、本地上传、样例预览、下载申请、审批与浏览器下载。
- 本地上传与审批结果通过浏览器本地存储保留，便于演示环境连续验证。

## 当前可用能力

- 可新建数据集记录，并立即出现在列表中。
- 可填写申请原因发起下载申请。
- 可审批下载申请，并把授权状态写回当前数据集。
- 审批后可触发浏览器下载本地生成的版本文件摘要。

## 当前限制

- 上传仍为前端本地模拟，不会真正写入对象存储或后端文件系统。
- 下载产物是本地生成的摘要文件，不是真实数据集压缩包。
- 后端若不可用，申请/审批接口会退化为本地演示记录。
- 因此当前状态应视为“可走通演示链路”，不是完整生产上传/下载系统。
