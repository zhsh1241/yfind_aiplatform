# Deep Interview Spec: F003-interactive-prototype

## 意图

先为 YFI 工业 AI 小模型平台产出一个可点击的前端原型，让项目干系人可以在浏览器中体验 MVP 主流程、模块入口和关键交互反馈。

## 期望结果

`frontend/` 中有一个无需真实后端数据即可运行的中文交互原型，覆盖导航、模块切换、详情抽屉、创建任务弹窗、训练步骤、部署确认、审计反馈等点击效果。

## 范围内

- React/Ant Design 单页交互原型。
- 组织权限、数据集、标注、训练、模型、推理、边缘、监控审计等 MVP 模块入口。
- 关键按钮点击后的弹窗、抽屉、步骤切换、状态提示和页面内容变化。
- 前端测试覆盖主要点击路径。

## 范围外

- 真实 API 调用、真实登录、真实文件上传、真实训练/部署。
- 后端业务实现或数据库表设计。
- 高保真视觉品牌设计和真实图标资产。
