# PRD: F003-interactive-prototype

## 问题

当前前端只有基础骨架，无法演示平台完整业务闭环。项目需要一个可点击原型来辅助业务评审、范围确认和后续功能拆分。

## 目标

- 用中文呈现平台 MVP 业务流程。
- 让所有主要导航和按钮都有可见点击反馈。
- 支持业务模块之间的快速切换和关键动作模拟。
- 不引入真实后端依赖，不填充未确认环境信息。

## 成功标准

- 前端首页变成完整平台原型而不是静态骨架。
- 主要模块点击后内容变化。
- 操作按钮触发抽屉、弹窗、步骤流、消息提示或状态更新。
- `npm run lint`、`npm run test:ci`、`npm run build` 通过。
