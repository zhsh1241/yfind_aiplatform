## 当前可用能力

- 平台总览首页已收敛为更稳定的生产控制台结构，可查看运行评分、核心指标、模块运行看板、值班摘要和链路入口。
- 组织权限页面支持发起授权登录申请、在审批工作台批准登录、自动切换到目标角色，并支持退出登录与恢复登录；页面排版已按生产后台风格收敛。
- 模块导航、流程节点、抽屉详情可正常使用。
- 数据资产支持本地新增数据集、发起下载申请、审批后触发浏览器下载。
- 训练中心支持通过弹窗提交流程，创建本地训练任务记录。
- 推理服务支持创建本地部署记录，并可执行健康检查确认、灰度放量与回滚到稳定版本。
- 边缘下发支持创建本地下发记录，并可确认同步完成、回滚到稳定包版本。
- 标注任务支持本地复核通过、退回补标与交付训练闭环。
- 监控审计支持本地告警确认、静默与升级 P1 闭环。
- 组织权限仍支持在本地查看当前权限集合和模块门禁矩阵。

## 当前边界

- 当前为“前端原型 + 本地模拟持久化 + 部分后端 API 联调”的混合态，不应描述为完整生产系统。
- 组织权限中的登录态、审批单和角色切换目前为本地持久化模拟，后续应替换为真实 IAM / 审批服务。
- 平台总览首页已删除部分噪音信息块，但总体数据仍以前端派生展示为主，后续可继续替换为后端实时数据源。
- 真实登录、文件对象存储、训练编排、KServe 发布、边缘 agent 下发、审计闭环仍需后续 feature 完成。
- Vite 构建提示主包超过 500 kB，原因是 Ant Design/图标打包；暂不阻断当前交付。
