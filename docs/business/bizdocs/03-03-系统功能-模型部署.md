# 03-03 系统功能清单 – 模型部署域

> **范围**：中心端推理服务、边端服务器管理等推理部署全流程
> **组织方式**：按子模块分组，每功能点含终端、模块路径、支撑场景、输入输出、约束

---

## 一、中心端推理服务

- **FUNC-INF-001 – 推理服务创建**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Service Creator
  - **支撑场景**：快速部署推理服务
  - **主要输入**：模型版本、服务参数、资源配置
  - **主要输出**：服务ID、部署配置
  - **约束**：选择模型版本，配置服务参数，Pr = M（Should Have）

- **FUNC-INF-002 – 协议支持**
  - **终端**：PC / Interface / Batch
  - **所属模块**：Inference Service / Protocol Support
  - **支撑场景**：多协议推理请求
  - **主要输入**：协议选择、请求内容
  - **主要输出**：推理响应
  - **约束**：支持HTTP、WebSocket、gRPC协议，Pr = M（Should Have）

- **FUNC-INF-003 – 服务手动上下线**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Service Lifecycle
  - **支撑场景**：服务部署控制
  - **主要输入**：服务ID、操作命令
  - **主要输出**：服务状态
  - **约束**：手动部署、下线控制，Pr = M（Should Have）

- **FUNC-INF-004 – 服务资源监控**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Resource Monitor
  - **支撑场景**：资源使用情况跟踪
  - **主要输入**：服务ID、时间范围
  - **主要输出**：资源使用统计、分配情况
  - **约束**：资源使用/分配情况总览，Pr = M（Should Have）

- **FUNC-INF-005 – 服务调用量监控**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Call Monitor
  - **支撑场景**：服务使用量统计
  - **主要输入**：服务ID、时间段
  - **主要输出**：调用数据、使用趋势
  - **约束**：24h内调用数据统计，Pr = M（Should Have）

- **FUNC-INF-006 – 访问权限管理**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Access Control
  - **支撑场景**：服务访问权限配置
  - **主要输入**：用户/应用、权限类型、期限
  - **主要输出**：API Key、访问权限
  - **约束**：API Key、访问时长配置，Pr = M（Should Have）

- **FUNC-INF-007 – 弹性扩缩容配置**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Elastic Scaling
  - **支撑场景**：自适应流量处理
  - **主要输入**：扩缩容规则、阈值
  - **主要输出**：扩缩容策略
  - **约束**：手动配置扩缩容规则，Pr = M（Should Have）

- **FUNC-INF-008 – 缩容到零**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Zero Scaling
  - **支撑场景**：资源成本优化
  - **主要输入**：缩容到零配置
  - **主要输出**：配置策略
  - **约束**：无请求时自动缩容到0，Pr = M（Should Have）

- **FUNC-INF-009 – 流量限制配置**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Rate Limiting
  - **支撑场景**：服务容量保护
  - **主要输入**：限流策略、阈值
  - **主要输出**：限流规则
  - **约束**：限流、防超载配置，Pr = M（Should Have）

- **FUNC-INF-010 – 服务终端调试**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Debug Terminal
  - **支撑场景**：故障排查
  - **主要输入**：服务ID
  - **主要输出**：Terminal会话
  - **约束**：打开服务容器Terminal进行调试，Pr = M（Should Have）

- **FUNC-INF-011 – 端口探活探测**
  - **终端**：PC / Interface / Batch
  - **所属模块**：Inference Service / Health Check
  - **支撑场景**：服务可用性检测
  - **主要输入**：服务ID、探测间隔
  - **主要输出**：健康检查结果
  - **约束**：服务健康检查，Pr = M（Should Have）

- **FUNC-INF-012 – 拨测验证**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Probe Verification
  - **支撑场景**：服务质量验证
  - **主要输入**：拨测配置、测试数据
  - **主要输出**：拨测结果、详情报告
  - **约束**：拨测详细结果、手动处理，Pr = M（Should Have）

- **FUNC-INF-013 – 安全围栏配置**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Safety Filter
  - **支撑场景**：内容安全防护
  - **主要输入**：过滤规则配置
  - **主要输出**：安全策略
  - **约束**：内容过滤规则配置，Pr = M（Should Have）

- **FUNC-INF-014 – 批量推理服务**
  - **终端**：PC / Interface / Batch
  - **所属模块**：Inference Service / Batch Inference
  - **支撑场景**：离线大批量处理
  - **主要输入**：数据源、推理任务配置
  - **主要输出**：推理结果、产物存储
  - **约束**：离线大批量数据处理，产出物保存，Pr = M（Should Have）

- **FUNC-INF-015 – 第三方推理服务纳管**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Third-party Integration
  - **支撑场景**：外部服务接入
  - **主要输入**：服务地址、认证信息
  - **主要输出**：服务配置、纳管状态
  - **约束**：HTTP/WS外部服务接入，Pr = M（Should Have）

## 三、批量推理与调度中心

- **FUNC-INF-025 – 批量推理任务提交**
  - **终端**：PC / Interface / Batch
  - **所属模块**：Batch Inference / Task Submitter
  - **支撑场景**：离线大批量数据处理
  - **主要输入**：输入数据集（上传或引用已有数据集）、目标推理服务、并发度、超时配置、调度时间
  - **主要输出**：批量任务 ID、执行计划
  - **约束**：支持立即执行和定时调度，Pr = M（Should Have）

- **FUNC-INF-026 – 批量调度策略配置**
  - **终端**：PC / Interface
  - **所属模块**：Batch Inference / Scheduler Config
  - **支撑场景**：批量任务公平调度与优先级管理
  - **主要输入**：调度策略选择（FIFO / PRIORITY_WEIGHT）、任务优先级权重（1~10）
  - **主要输出**：策略生效确认、任务队列状态
  - **约束**：支持 FIFO（先进先出）和 PRIORITY_WEIGHT（优先级加权）两种全局调度策略；管理员可随时 toggle 启用/停用；PRIORITY_WEIGHT 模式下任务可设置 1~10 优先级权重，Pr = M（Should Have）

- **FUNC-INF-027 – 批量任务进度监控**
  - **终端**：PC / Interface
  - **所属模块**：Batch Inference / Progress Monitor
  - **支撑场景**：批量任务执行状态跟踪
  - **主要输入**：批量任务 ID
  - **主要输出**：进度指标（已处理/总数/失败数）、ETA、吞吐量 QPS、失败样本详情
  - **约束**：实时展示进度；失败样本支持查看详情和重试，Pr = M（Should Have）

- **FUNC-INF-028 – 批量推理结果下载/推送**
  - **终端**：PC / Interface
  - **所属模块**：Batch Inference / Result Exporter
  - **支撑场景**：批量推理结果交付
  - **主要输入**：结果格式选择（CSV/JSON/Parquet）、对象存储目标（OSS/S3）、Webhook 地址
  - **主要输出**：结果文件/推送任务、Webhook 通知
  - **约束**：支持下载为 CSV/JSON/Parquet；支持异步推送至指定对象存储（OSS/S3）；推送完成后 Webhook 通知，Pr = M（Should Have）

- **FUNC-INF-029 – 推理服务扩缩容配置细节**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Auto Scaling Config
  - **支撑场景**：弹性资源调度与成本优化
  - **主要输入**：扩缩容策略（HPA / SERVERLESS）、CPU/GPU 利用率阈值、最小/最大副本数
  - **主要输出**：扩缩容策略配置、副本数变化记录
  - **约束**：HPA 基于 CPU/GPU 利用率阈值自动扩缩；SERVERLESS 请求量为零时缩至 0 副本，冷启动 < 5s；支持设置最小/最大副本数，Pr = M（Should Have）

- **FUNC-INF-030 – 容器健康探针配置**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Health Probe Config
  - **支撑场景**：推理服务可用性保障
  - **主要输入**：探针类型（TCP 端口 / HTTP 路径）、初始化等待时间、检测间隔、失败次数阈值
  - **主要输出**：健康探针配置、容器健康状态
  - **约束**：支持 TCP 端口探针和 HTTP 路径探针（/health）；可配置初始化等待时间、检测间隔、失败次数阈值，Pr = M（Should Have）

- **FUNC-INF-031 – 第三方推理服务纳管管理**
  - **终端**：PC / Interface
  - **所属模块**：Inference Service / Third-party Service Manager
  - **支撑场景**：外部推理服务统一管理与续期
  - **主要输入**：访问配置（地址/认证）、availableUntil 续期时间
  - **主要输出**：服务状态列表（激活/到期/停用）、连接测试结果
  - **约束**：支持编辑访问配置、手动触发连接测试、续期 availableUntil；到期前 7 天标黄提醒（对应规则 INF-010），Pr = M（Should Have）

---

> **约束**：所有清单仅基于 SOR V1.0 内容抽取；优先级 H=Must Have，M=Should Have，L=Could Live Without。
