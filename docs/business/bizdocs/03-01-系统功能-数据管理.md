# 03-01 系统功能清单 – 数据管理域

> **范围**：数据源接入、数据加工预处理、数据标注、数据集管理、数据增强、数据集门户等全流程
> **组织方式**：按子模块分组，每功能点含终端、模块路径、支撑场景、输入输出、约束

---

## 一、数据源管理

- **FUNC-DATA-001 – 数据源接入**
  - **终端**：PC / Interface
  - **所属模块**：DataSource Management / Source Connector
  - **支撑场景**：多源数据统一纳管
  - **主要输入**：关系型DB（MySQL/PostgreSQL/Oracle）、文件（CSV/JSON/Parquet）、对象存储（OSS/S3）、流媒体、时序数据库、工业协议（Modbus/OPC-UA）、REST API等
  - **主要输出**：数据源配置、连接信息、元数据
  - **约束**：支持团队共享公开模式，Pr = H（Must Have）

- **FUNC-DATA-002 – 数据源连接测试**
  - **终端**：PC / Interface
  - **所属模块**：DataSource Management / Connection Validator
  - **支撑场景**：验证数据源可连通性
  - **主要输入**：数据源连接参数
  - **主要输出**：连接测试结果、连通性状态
  - **约束**：Pr = H（Must Have）

- **FUNC-DATA-003 – 数据源权限配置**
  - **终端**：PC / Interface
  - **所属模块**：DataSource Management / Permission Control
  - **支撑场景**：团队数据源共享权限管理
  - **主要输入**：用户/团队、权限类型（读/写/删除）
  - **主要输出**：权限策略、访问控制列表
  - **约束**：支持团队级权限隔离，Pr = H（Must Have）

- **FUNC-DATA-004 – 数据源状态监控**
  - **终端**：PC / Interface
  - **所属模块**：DataSource Management / Health Monitor
  - **支撑场景**：数据源实时状态跟踪
  - **主要输入**：数据源ID
  - **主要输出**：连接状态、采集状态、最后同步时间
  - **约束**：Pr = M（Should Have）

- **FUNC-DATA-005 – 数据采集任务配置**
  - **终端**：PC / Interface
  - **所属模块**：DataSource Management / Collection Task
  - **支撑场景**：可视化配置多种数据采集需求
  - **主要输入**：数据源、数据类型（文本/图片/音频/视频）、采集规则、调度参数
  - **主要输出**：采集任务配置、执行计划
  - **约束**：支持文本、图片、音频、视频等多模态采集，Pr = H（Must Have）

  - **约束**：AI 推荐不强制覆盖用户已配置参数，Pr = M（Should Have）

- **FUNC-DATA-085 – Pipeline 版本快照**

  - **所属模块**：Data Pipeline / Version Control
  - **支撑场景**：Pipeline 迭代管理与回滚
  - **主要输入**：保存操作触发
  - **主要输出**：版本快照记录、版本间 diff 视图
  - **约束**：每次保存生成版本快照，支持回滚至任意历史版本，Pr = M（Should Have）
