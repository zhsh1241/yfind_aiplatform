## 一、平台运营配置

> 本节覆盖数据集门户展示开关、上传下载限制、标注配置等运营参数，属于超级管理员/BU子管理员的日常配置操作，不涉及认证与权限逻辑。

  - **约束**：按名称、标签检索镜像，Pr = M（Should Have）

- **FUNC-RES-037 – 国产芯片架构支持**
  - **终端**：PC / Interface
  - **所属模块**：Image Management / Chip Compatibility
  - **支撑场景**：国产化适配与部署
  - **主要输入**：目标架构（ARM64 / X86）、芯片型号（华为昇腾 910B2 等）
  - **主要输出**：架构兼容镜像、适配报告
  - **约束**：镜像支持 ARM64 物理框架标注；支持华为昇腾 910B2 等国产 AI 芯片 Driver 版本字段；构建时自动检测架构兼容性，Pr = M（Should Have）

- **FUNC-RES-038 – 三层资源监控视图（K8s / YARN / 主机）**
  - **终端**：PC / Interface
  - **所属模块**：Resource Monitoring / Multi-Layer Monitor
  - **支撑场景**：异构集群资源全景观测
  - **主要输入**：集群 ID、时间范围、层级选择
  - **主要输出**：K8s Namespace 级 CPU/GPU/内存趋势图；YARN 队列资源占用表；主机级 CPU/内存/磁盘/网络负载折线图
  - **约束**：K8s 层显示 Namespace 级资源趋势（近 1h/6h/24h）；YARN 层显示队列 Pending/Running/Completed 任务数及资源占比；主机层以卡片形式展示每台物理机核心指标，Pr = H（Must Have）

- **FUNC-RES-039 – 调度中心全局策略配置**
  - **终端**：PC / Interface
  - **所属模块**：Scheduling Center / Strategy Config
  - **支撑场景**：训练/推理任务队列策略管理
  - **主要输入**：调度策略选择（FIFO / PRIORITY_WEIGHT）、启用/停用状态
  - **主要输出**：策略配置记录、生效时间
  - **约束**：支持 FIFO（先进先出，任务按提交时间排队）和 PRIORITY_WEIGHT（按优先级权重 1~10 动态调度）两种全局策略；管理员可随时切换，切换立即对新进入队列任务生效，已排队任务不受影响；提供启用/停用 Toggle，Pr = M（Should Have）

  - **主要输入**：各配置项数值（见约束）
  - **主要输出**：全局安全策略、生效配置快照
  - **约束**：

    - 单文件大小上限（默认 200 MB）
    - 数据集样本数上限（默认 100,000 条/数据集版本）
    - 以上均可在 BU 级覆盖（BU Admin 可在不超过集团上限的范围内收紧），Pr = H（Must Have）
