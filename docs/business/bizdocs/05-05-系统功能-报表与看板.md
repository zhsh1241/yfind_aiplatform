# 05-05 系统功能清单 – 报表与运营看板

> **范围**：平台运营看板、业务报表、报表通用能力等运营统计和数据分析
> **组织方式**：按子模块分组，每功能点含终端、模块路径、支撑场景、输入输出、约束
> **对应场景**：SYS-009

---

## 一、平台运营看板

- **FUNC-REPORT-001 – 平台总览看板**
  - **终端**：PC / Interface
  - **所属模块**：Dashboard / Platform Overview
  - **支撑场景**：平台整体运营状态一览
  - **主要输入**：时间范围、维度筛选（BU等）
  - **主要输出**：统计卡片、趋势图表、占比分布
  - **约束**：全平台统计（数据集总数/总存储量/模型总数/活跃用户数/运行中任务数）；各BU占比饼图；本月新增趋势折线图；按BU/时间维度切换；Pr = H（Must Have）

- **FUNC-REPORT-002 – 数据资产看板**
  - **终端**：PC / Interface
  - **所属模块**：Dashboard / Data Assets
  - **支撑场景**：数据资产状态监控与运营决策
  - **主要输入**：时间范围、数据类型筛选
  - **主要输出**：数据集统计、分布图表、热门排名
  - **约束**：数据集数量/存储量/下载量/标注量趋势；按数据类型（图片/文本/音频/视频）分布；热门数据集Top10；标注效率统计（人均日标注量）；Pr = H（Must Have）

- **FUNC-REPORT-003 – 模型资产看板**
  - **终端**：PC / Interface
  - **所属模块**：Dashboard / Model Assets
  - **支撑场景**：模型资产运营状态跟踪
  - **主要输入**：时间范围、模型类型筛选
  - **主要输出**：模型统计、框架分布、使用排名
  - **约束**：模型总数/版本数/按框架分布/按任务类型分布；模型调优任务成功率/平均耗时；模型下发边端统计；热门基础模型使用排名；Pr = H（Must Have）

- **FUNC-REPORT-004 – 资源使用看板**
  - **终端**：PC / Interface
  - **所属模块**：Dashboard / Resource Usage
  - **支撑场景**：计算资源成本与效能分析
  - **主要输入**：时间范围、资源池筛选
  - **主要输出**：使用率曲线、分布图表、浪费分析
  - **约束**：GPU/CPU/内存/存储使用率实时+历史趋势；按BU/资源池分布；峰值使用时段热力图；资源空闲浪费分析；资源利用率同比/环比；Pr = H（Must Have）

- **FUNC-REPORT-005 – 任务执行看板**
  - **终端**：PC / Interface
  - **所属模块**：Dashboard / Task Execution
  - **支撑场景**：任务系统运营效能监控
  - **主要输入**：时间范围、任务类型筛选
  - **主要输出**：任务统计、成功率分析、耗时分布
  - **约束**：训练任务/调优任务/标注任务数量趋势；任务成功率/失败原因分布；平均排队时长/执行时长；GPU利用率与任务相关性分析；Pr = M（Should Have）

- **FUNC-REPORT-006 – 用户活跃度看板**
  - **终端**：PC / Interface
  - **所属模块**：Dashboard / User Activity
  - **支撑场景**：用户增长与产品活跃度分析
  - **主要输入**：时间范围、用户角色筛选
  - **主要输出**：活跃度曲线、功能热力图、用户路径
  - **约束**：DAU/MAU趋势；按角色活跃用户分布；各功能模块使用频率热力图；新用户增长趋势；用户操作路径Top10；Pr = M（Should Have）

## 二、业务报表

- **FUNC-REPORT-010 – 数据集统计报表**
  - **终端**：PC / Interface
  - **所属模块**：Reports / Dataset Report
  - **支撑场景**：数据资产运营与项目成本统计
  - **主要输入**：时间段、BU/项目筛选
  - **主要输出**：Excel报告、邮件通知
  - **约束**：按BU/项目/时间段汇总（数据集数量/存储量/标注完成率/增强数量）；支持导出Excel；定时邮件报告；Pr = H（Must Have）

- **FUNC-REPORT-011 – 标注工作量报表**
  - **终端**：PC / Interface
  - **所属模块**：Reports / Annotation Report
  - **支撑场景**：标注团队绩效考核与质量管理
  - **主要输入**：时间段、标注工程师/场景/数据集筛选
  - **主要输出**：Excel报告、绩效数据
  - **约束**：按标注工程师统计（标注数量/审核通过率/标注效率（条/小时））；按时间段/标注场景/数据集维度筛选；支持绩效考核导出；Pr = H（Must Have）

- **FUNC-REPORT-012 – 模型训练报表**
  - **终端**：PC / Interface
  - **所属模块**：Reports / Training Report
  - **支撑场景**：模型开发成本核算与项目管理
  - **主要输入**：时间段、BU/模型类型筛选
  - **主要输出**：Excel报告、资源消耗汇总
  - **约束**：训练任务明细（任务ID/模型/数据集/超参数/资源消耗/耗时/最终指标）；按BU/模型类型/时间筛选；GPU小时消耗统计；Pr = H（Must Have）

- **FUNC-REPORT-013 – 推理服务使用报表**
  - **终端**：PC / Interface
  - **所属模块**：Reports / Inference Report
  - **支撑场景**：推理服务成本分摊与性能分析
  - **主要输入**：时间段、服务/调用方筛选
  - **主要输出**：Excel报告、服务质量指标
  - **约束**：推理服务调用量/错误率/平均延迟/P95延迟趋势；按服务/调用方/时间维度；API Key使用量明细；Pr = M（Should Have）

- **FUNC-REPORT-014 – 资源消耗报表**
  - **终端**：PC / Interface
  - **所属模块**：Reports / Resource Report
  - **支撑场景**：云计算成本分摊与部门预算管理
  - **主要输入**：时间段（周/月/季度）、BU/项目筛选
  - **主要输出**：Excel报告、成本数据
  - **约束**：各BU/项目的CPU/GPU/存储资源消耗明细；用于内部成本分摊；按周/月/季度汇总；支持导出；Pr = H（Must Have）

- **FUNC-REPORT-015 – 边端服务运行报表**
  - **终端**：PC / Interface
  - **所属模块**：Reports / Edge Server Report
  - **支撑场景**：边端基础设施运维与SLA评估
  - **主要输入**：时间段、边端服务器筛选
  - **主要输出**：Excel报告、运行指标
  - **约束**：各边端服务器统计（在线率/模型版本/推理调用量/资源使用）；下发任务成功率/失败原因；Pr = M（Should Have）

- **FUNC-REPORT-016 – 安全与合规报表**
  - **终端**：PC / Interface
  - **所属模块**：Reports / Security Report
  - **支撑场景**：安全运营与合规审计
  - **主要输入**：时间段（月/季/年）筛选
  - **主要输出**：PDF报告、合规证明
  - **约束**：权限变更汇总/高风险操作统计/数据下载量统计/未授权访问尝试；按月/季/年汇总；用于安全合规审查；Pr = M（Should Have）

## 三、报表通用能力

- **FUNC-REPORT-020 – 报表过滤与下钻**
  - **终端**：PC / Interface
  - **所属模块**：Reports / Advanced Filtering
  - **支撑场景**：深度数据分析
  - **主要输入**：过滤条件、下钻维度
  - **主要输出**：筛选后数据、明细表格
  - **约束**：所有报表支持时间范围/BU/项目多维过滤；支持图表下钻（点击BU展开项目明细）；Pr = H（Must Have）

- **FUNC-REPORT-021 – 报表导出**
  - **终端**：PC / Interface
  - **所属模块**：Reports / Export Manager
  - **支撑场景**：报表数据外部使用与存档
  - **主要输入**：导出格式选择、导出范围
  - **主要输出**：Excel/CSV/PDF文件、下载链接
  - **约束**：支持导出Excel/CSV/PDF；大数据量导出异步处理+下载链接通知；导出操作记录审计；Pr = H（Must Have）

- **FUNC-REPORT-022 – 定时报告推送**
  - **终端**：PC / Interface
  - **所属模块**：Reports / Scheduled Reporter
  - **支撑场景**：自动化运营报告分发
  - **主要输入**：报告配置、收件人、推送频率
  - **主要输出**：邮件/IM消息、PDF附件
  - **约束**：管理员配置定时报告（选择报表类型/收件人/推送频率（日/周/月）/推送渠道（邮件/IM））；生成PDF附件；Pr = M（Should Have）

- **FUNC-REPORT-023 – 自定义看板**
  - **终端**：PC / Interface
  - **所属模块**：Dashboard / Customization
  - **支撑场景**：个性化运营指标监控
  - **主要输入**：看板配置、关注指标
  - **主要输出**：个人看板、共享模板
  - **约束**：用户可拖拽配置个人看板（选择关注的统计卡片/图表）；支持保存多个看板配置；超级管理员可发布组织看板模板；Pr = M（Should Have）

- **FUNC-REPORT-024 – 数据刷新策略**
  - **终端**：API / 后端
  - **所属模块**：Reports / Data Refresh
  - **支撑场景**：性能与数据实时性平衡
  - **主要输入**：刷新策略配置
  - **主要输出**：最新数据、缓存状态
  - **约束**：实时看板数据≤1分钟延迟；历史报表T+1更新；支持手动刷新；缓存策略避免频繁查询影响业务；Pr = M（Should Have）

---

> **约束**：所有清单仅基于 SOR V1.0 内容抽取；优先级 H=Must Have，M=Should Have，L=Could Live Without。
