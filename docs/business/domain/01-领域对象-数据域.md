# 01-领域对象 – 数据域

> **范围**：数据域负责数据源接入、数据加工、标注任务、数据集版本管理和数据增强处理。  
> **组织方式**：按聚合根分组。每个聚合根包含标识、核心属性、状态机、关联关系和子对象。

---

## 聚合根1：数据源（DataSource Aggregate）

**领域对象**：`DataSource`  
**标识**：`sourceId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `sourceId` | UUID | 数据源唯一标识 |
| `name` | String | 数据源名称 |
| `type` | Enum | 数据源类型：`RELATIONAL_DB` / `FILE` / `OBJECT_STORAGE` / `STREAM` / `TIME_SERIES` / `INDUSTRIAL_PROTOCOL` / `API` |
| `connectionConfig` | Object | 连接配置（主机、端口、认证信息等） |
| `description` | String | 数据源描述 |
| `status` | Enum | 状态（见下方状态机） |
| `teamSharing` | Boolean | 是否团队内共享 |
| `createdBy` | UUID | 创建者ID |
| `tenantId` | UUID | 所属租户ID |
| `createdAt` | Timestamp | 创建时间 |
| `updatedAt` | Timestamp | 更新时间 |

**状态机**：
```
未激活 → 活跃 → 连接失败 → 禁用
           ↓
         连接失败
```

- **未激活**：数据源已创建，未进行过连接测试
- **活跃**：连接正常，可用于后续操作
- **连接失败**：最近一次连接测试失败，自动转为此状态
- **禁用**：被管理员或用户主动禁用

**关联关系**：
- `1:N → DataPipeline`：一个数据源可被多个数据加工流水线引用
- `1:N → AnnotationTask`：一个数据源可作为多个标注任务的数据来源

**操作**：
- `testConnection()`：测试连接有效性
- `updateConnectionConfig(config)`：更新连接配置
- `changeStatus(newStatus)`：转换状态

## 聚合根2：数据加工流水线（DataPipeline Aggregate）

**领域对象**：`DataPipeline`  
**标识**：`pipelineId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `pipelineId` | UUID | 流水线唯一标识 |
| `name` | String | 流水线名称 |
| `description` | String | 流水线描述 |
| `sourceDatasetId` | UUID | 源数据集ID（引用Dataset） |
| `steps` | List<PipelineStep> | 算子列表（有序） |
| `outputDatasetId` | UUID | 输出数据集ID |
| `status` | Enum | 状态（见下方状态机） |
| `config` | Object | 流水线配置（超时、重试、并发度等） |
| `createdBy` | UUID | 创建者ID |
| `tenantId` | UUID | 所属租户ID |
| `createdAt` | Timestamp | 创建时间 |
| `updatedAt` | Timestamp | 更新时间 |

**子对象**：`PipelineStep`

| 属性 | 类型 | 说明 |
|------|------|------|
| `stepId` | String | 步骤标识 |
| `operator` | String | 算子类型（过滤、转换、聚合、清洗等） |
| `parameters` | Object | 算子参数 |
| `order` | Integer | 执行顺序 |

**状态机**：
```
草稿 → 运行中 → 已完成
  ↓       ↓
失败   已取消
```

- **草稿**：流水线已创建，未执行
- **运行中**：流水线正在执行
- **已完成**：流水线执行成功，输出数据集已生成
- **失败**：流水线执行失败
- **已取消**：用户取消流水线执行

**关联关系**：
- `N:1 → DataSource`：流水线关联一个源数据源
- `1:1 → Dataset`：流水线输出关联一个输出数据集

**操作**：
- `addStep(operator, parameters)`：添加处理步骤
- `removeStep(stepId)`：移除步骤
- `execute()`：执行流水线
- `cancel()`：取消执行

## 聚合根3：标注任务（AnnotationTask Aggregate）

**领域对象**：`AnnotationTask`  
**标识**：`taskId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `taskId` | UUID | 任务唯一标识 |
| `name` | String | 任务名称 |
| `scene` | Enum | 标注场景：`IMAGE_CLASSIFICATION` / `OBJECT_DETECTION` / `SEMANTIC_SEGMENTATION` / `TEXT_LABELING` / `AUDIO_LABELING` / `VIDEO_LABELING` / `MULTI_MODAL` |
| `labelTemplate` | Object | 引用LabelTemplate，包含标签体系 |
| `assignees` | List<UUID> | 标注员ID列表 |
| `sourceDatasetId` | UUID | 数据来源数据集ID |
| `annotatedCount` | Integer | 已标注数据条数 |
| `reviewedCount` | Integer | 已审核通过数据条数 |
| `status` | Enum | 状态（见下方状态机） |
| `priority` | Enum | 优先级：HIGH / MEDIUM / LOW |
| `startTime` | Timestamp | 任务开始时间 |
| `deadline` | Timestamp | 任务截止时间 |
| `createdBy` | UUID | 创建者ID |
| `tenantId` | UUID | 所属租户ID |
| `createdAt` | Timestamp | 创建时间 |
| `updatedAt` | Timestamp | 更新时间 |

**状态机**：
```
创建 → 标注中 → 待审核 → 审核通过 → 已完成
                  ↓
                驳回 ──→ 标注中
```

- **创建**：任务已创建，未分配标注员
- **标注中**：标注员正在进行标注
- **待审核**：标注完成，等待审核
- **审核通过**：审核通过，数据可用
- **驳回**：审核不通过，返回标注环节
- **已完成**：标注任务完全结束

**关联关系**：
- `1:1 → LabelTemplate`：每个标注任务关联一个标签模板
- `N:1 → Dataset`：标注任务的数据来源是某个数据集
- `N:N → User（标注员）`：通过assignees关联

**操作**：
- `assignAnnotator(userId)`：分配标注员
- `submitAnnotation(dataId, labels)`：提交标注数据
- `submitForReview()`：提交审核
- `approve()`：审核通过
- `reject(reason)`：审核驳回

## 聚合根4：数据集（Dataset Aggregate）

**领域对象**：`Dataset`  
**标识**：`datasetId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `datasetId` | UUID | 数据集唯一标识 |
| `name` | String | 数据集名称 |
| `type` | Enum | 数据集类型：`RAW` / `PREPROCESSED` / `ANNOTATED` / `AUGMENTED` |
| `dataType` | Enum | 数据内容类型：`TEXT` / `IMAGE` / `AUDIO` / `VIDEO` / `MULTI_MODAL` |
| `size` | Long | 数据集总大小（字节） |
| `recordCount` | Long | 数据记录条数 |
| `version` | String | 当前版本号 |
| `status` | Enum | 状态（见下方状态机） |
| `lineage` | List<UUID> | 数据血缘：上游数据集ID列表 |
| `tags` | List<String> | 数据集标签（用于分类和检索） |
| `accessLevel` | Enum | 访问级别：`PUBLIC` / `TEAM` / `PRIVATE` |
| `storageLocation` | String | 存储位置（路径或URL） |
| `owner` | UUID | 所有者ID |
| `tenantId` | UUID | 所属租户ID |
| `createdAt` | Timestamp | 创建时间 |
| `updatedAt` | Timestamp | 更新时间 |

**子对象**：`DatasetVersion`

| 属性 | 类型 | 说明 |
|------|------|------|
| `versionId` | UUID | 版本ID |
| `versionNo` | String | 版本号（如v1.0, v1.1） |
| `recordCount` | Long | 该版本的记录数 |
| `size` | Long | 该版本的大小 |
| `storageLocation` | String | 该版本的存储位置 |
| `description` | String | 版本描述 |
| `createdAt` | Timestamp | 创建时间 |

**子对象**：`DatasetTag`

| 属性 | 类型 | 说明 |
|------|------|------|
| `tagId` | UUID | 标签ID |
| `tagName` | String | 标签名称 |
| `category` | String | 标签分类 |

**子对象**：`DataLineage`

| 属性 | 类型 | 说明 |
|------|------|------|
| `lineageId` | UUID | 血缘记录ID |
| `sourceDatasetId` | UUID | 上游数据集ID |
| `targetDatasetId` | UUID | 下游数据集ID |
| `transformType` | String | 转换类型（如：pipeline, annotation, augmentation） |
| `createdAt` | Timestamp | 创建时间 |

**状态机**：
```
草稿 → 发布 → 已归档 → 已回收
         ↓
      禁用
```

- **草稿**：数据集已创建，未发布
- **发布**：数据集已发布，可被引用和使用
- **已归档**：数据集已归档，不接受新操作
- **禁用**：数据集被禁用，不可访问
- **已回收**：数据集已回收，逻辑删除

**关联关系**：
- `1:N → DatasetVersion`：一个数据集可有多个版本
- `1:N → DatasetTag`：一个数据集可有多个标签
- `1:N → DataLineage`：记录数据集的血缘关系
- `M:N → AnnotationTask`：通过annotation关联
- `M:N → FineTuningJob`：作为训练数据源

**操作**：
- `createVersion(description)`：创建新版本
- `addTag(tagName, category)`：添加标签
- `publish()`：发布数据集
- `archive()`：归档数据集
- `recycle()`：回收数据集
- `getLineage()`：获取血缘信息

## 聚合根7：数据集访问申请（AccessRequest Aggregate）

**领域对象**：`AccessRequest`  
**标识**：`requestId: UUID`  
**背景**：对应 DAT-006 规则——受限（Restricted）级别数据集的访问须经审批。

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `requestId` | UUID | 申请唯一标识 |
| `datasetId` | UUID | 申请访问的数据集ID |
| `applicantId` | UUID | 申请人用户ID |
| `applicantTenantId` | UUID | 申请人所属租户ID |
| `accessLevel` | Enum | 申请权限：`READ` / `READ_WRITE` |
| `reason` | String | 申请理由 |
| `status` | Enum | 状态（见下方状态机） |
| `expiresAt` | Timestamp | 申请授权到期时间（审批通过时设置） |
| `reviewedBy` | UUID | 审批人ID（null 表示未审批） |
| `reviewComment` | String | 审批意见 |
| `reviewedAt` | Timestamp | 审批时间 |
| `autoRejectedAt` | Timestamp | 自动拒绝时间（超时48h后填充） |
| `createdAt` | Timestamp | 申请创建时间 |

**状态机**：
```
待审批 → 已通过 → 已过期
   ↓
 已拒绝（人工）
   ↓
 已拒绝（超时自动，48h）
```

- **待审批**：申请已提交，等待数据集所有者或BU管理员审核
- **已通过**：审批人批准，访问权限生效至 `expiresAt`
- **已拒绝（人工）**：审批人明确拒绝，附带拒绝原因
- **已拒绝（超时）**：48小时内无人审批，系统自动拒绝（规则 DAT-006）
- **已过期**：审批通过的权限已到期，自动撤销访问 ACL

**关联关系**：
- `N:1 → Dataset`：针对某个数据集的访问申请
- `N:1 → User (applicant)`：申请人
- `1:1 → ApprovalTicket`：关联审批工单（记录完整审批流程）

**操作**：
- `submit()`：提交申请
- `approve(reviewerId, expiresAt, comment)`：审批通过
- `reject(reviewerId, comment)`：人工拒绝
- `autoReject()`：系统定时任务触发，超时自动拒绝
- `expire()`：权限到期，自动撤销

## 聚合根8：审批工单（ApprovalTicket Aggregate）

**领域对象**：`ApprovalTicket`  
**标识**：`ticketId: UUID`  
**背景**：记录受限数据集访问审批的完整审批流程与操作历史，供审计追踪使用。

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `ticketId` | UUID | 工单唯一标识 |
| `requestId` | UUID | 关联的访问申请ID |
| `datasetId` | UUID | 目标数据集ID |
| `applicantId` | UUID | 申请人ID |
| `assignedTo` | UUID | 当前指派的审批人ID（数据集Owner或BU Admin） |
| `deadline` | Timestamp | 审批截止时间（`createdAt` + 48h） |
| `priority` | Enum | 优先级：`NORMAL` / `URGENT` |
| `history` | List<ApprovalEvent> | 审批事件列表（含通知、审批、超时等） |
| `status` | Enum | 工单状态：`OPEN` / `CLOSED` |
| `createdAt` | Timestamp | 工单创建时间 |
| `closedAt` | Timestamp | 工单关闭时间 |

**子对象 — 审批事件（ApprovalEvent）**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `eventId` | UUID | 事件ID |
| `eventType` | Enum | 事件类型：`SUBMITTED` / `NOTIFIED` / `APPROVED` / `REJECTED` / `TIMEOUT_REJECTED` / `EXPIRED` |
| `operatorId` | UUID | 操作人ID（系统自动操作时为 `SYSTEM`） |
| `comment` | String | 备注 |
| `occurredAt` | Timestamp | 事件发生时间 |

**关联关系**：
- `1:1 → AccessRequest`：一个工单对应一个访问申请
- `N:1 → User (assignedTo)`：被指派的审批人

**操作**：
- `addEvent(eventType, operatorId, comment)`：追加审批事件
- `close()`：关闭工单

## 跨聚合根关系总结

| 源聚合根 | 关联 | 目标聚合根 | 说明 |
|---------|------|-----------|------|
| DataSource | 1:N | DataPipeline | 数据源被流水线引用 |
| DataSource | 1:N | AnnotationTask | 数据源用于标注 |
| DataPipeline | N:1 | DataSource | 流水线的数据来源 |
| DataPipeline | 1:1 | Dataset | 流水线的输出结果 |
| AnnotationTask | 1:1 | LabelTemplate | 标注模板 |
| AnnotationTask | N:1 | Dataset | 标注的数据来源 |
| Dataset | 1:N | DatasetVersion | 版本管理 |
| Dataset | 1:N | DatasetTag | 标签管理 |
| Dataset | 1:N | DataLineage | 血缘追踪 |
| DataAugmentationJob | N:1 | Dataset | 源数据集 |
| DataAugmentationJob | 1:1 | Dataset | 输出数据集 |
| AccessRequest | N:1 | Dataset | 受限数据集访问申请 |
| AccessRequest | 1:1 | ApprovalTicket | 对应审批工单 |
| ApprovalTicket | 1:1 | AccessRequest | 记录审批流程 |
