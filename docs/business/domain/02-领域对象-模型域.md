# 02-领域对象 – 模型域

> **范围**：模型域负责模型调优、模型版本管理、模型工程化、开发环境和训练任务管理。  
> **组织方式**：按聚合根分组。每个聚合根包含标识、核心属性、状态机、关联关系和子对象。

---

## 聚合根1：模型调优任务（FineTuningJob Aggregate）

**领域对象**：`FineTuningJob`  
**标识**：`jobId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `jobId` | UUID | 任务唯一标识 |
| `name` | String | 任务名称 |
| `description` | String | 任务描述 |
| `baseModelId` | UUID | 基础模型ID（引用Model） |
| `datasetId` | UUID | 训练数据集ID |
| `trainSplit` | Double | 训练集比例（0-1） |
| `validationSplit` | Double | 验证集比例（0-1） |
| `testSplit` | Double | 测试集比例（0-1） |
| `hyperparameters` | List<HyperParameter> | 超参配置列表 |
| `resourceConfig` | ResourceConfig | 资源配置 |
| `status` | Enum | 状态（见下方状态机） |
| `outputModelId` | UUID | 输出模型ID |
| `logs` | String | 训练日志地址（文件路径或URL） |
| `createdBy` | UUID | 创建者ID |
| `tenantId` | UUID | 所属租户ID |
| `createdAt` | Timestamp | 创建时间 |
| `startedAt` | Timestamp | 开始时间 |
| `completedAt` | Timestamp | 完成时间 |

**子对象**：`HyperParameter`

| 属性 | 类型 | 说明 |
|------|------|------|
| `paramId` | String | 超参ID |
| `paramName` | String | 超参名称（如learning_rate、batch_size） |
| `value` | Any | 超参值 |
| `type` | Enum | 超参类型：FLOAT / INT / STRING / BOOL |

**子对象**：`TrainingMetric`

| 属性 | 类型 | 说明 |
|------|------|------|
| `metricId` | UUID | 指标ID |
| `metricName` | String | 指标名称（accuracy, loss, F1等） |
| `value` | Double | 指标值 |
| `timestamp` | Timestamp | 采样时间 |
| `epoch` | Integer | 对应的训练轮次 |

**子对象**：`ResourceConfig`

| 属性 | 类型 | 说明 |
|------|------|------|
| `cpu` | String | CPU规格（如"4"表示4核） |
| `gpu` | String | GPU规格（如"2"表示2张，或"2×A100"） |
| `npu` | String | NPU规格 |
| `memory` | String | 内存大小（如"16Gi"） |
| `storageSize` | String | 临时存储大小 |

**状态机**：
```
待提交 → 等待调度 → 运行中 → 已完成
           ↓        ↓
         失败   已取消
```

- **待提交**：任务已创建，未提交执行
- **等待调度**：任务已提交，等待资源调度
- **运行中**：任务正在训练
- **已完成**：训练完成，输出模型已生成
- **失败**：训练失败
- **已取消**：用户取消任务

**关联关系**：
- `N:1 → Model`：基础模型
- `N:1 → Dataset`：训练数据集
- `1:1 → Model`：输出模型（通过outputModelId）

**操作**：
- `submitJob()`：提交训练任务
- `cancel()`：取消任务
- `getMetrics()`：获取训练指标
- `getLogs()`：获取训练日志

## 聚合根2：模型（Model Aggregate）

**领域对象**：`Model`  
**标识**：`modelId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `modelId` | UUID | 模型唯一标识 |
| `name` | String | 模型名称 |
| `description` | String | 模型描述 |
| `framework` | Enum | 框架：PYTORCH / TENSORFLOW / PADDLE / ONNX / MXNET |
| `taskType` | Enum | 任务类型：IMAGE_CLASSIFICATION / OBJECT_DETECTION / SEMANTIC_SEGMENTATION / NLP_TEXT_CLASSIFICATION / NLP_NER / NLP_QA / TIME_SERIES_FORECAST / ANOMALY_DETECTION |
| `inputFormat` | String | 输入格式说明（如图片尺寸、文本长度等） |
| `outputFormat` | String | 输出格式说明（如分类标签、检测框等） |
| `sourceType` | Enum | 来源：TRAINING_ARTIFACT / USER_UPLOAD / PRETRAINED |
| `scope` | Enum | 使用范围：PLATFORM / TEAM / PRIVATE |
| `status` | Enum | 状态（见下方状态机） |
| `currentVersionId` | UUID | 当前活跃版本ID |
| `owner` | UUID | 所有者ID |
| `tenantId` | UUID | 所属租户ID |
| `createdAt` | Timestamp | 创建时间 |
| `updatedAt` | Timestamp | 更新时间 |

**子对象**：`ModelVersion`

| 属性 | 类型 | 说明 |
|------|------|------|
| `versionId` | UUID | 版本ID |
| `versionNo` | String | 版本号（如v1.0, v1.1） |
| `filePath` | String | 模型文件存储路径 |
| `fileSize` | Long | 模型文件大小 |
| `metrics` | Object | 性能指标（精度、速度等） |
| `status` | Enum | 版本状态：DRAFT / RELEASED / ARCHIVED |
| `createdAt` | Timestamp | 创建时间 |
| `createdBy` | UUID | 创建者ID |
| `runtimeRequirements` | Object | 运行时要求（框架版本、CUDA版本等） |

**状态机**：
```
草稿 → 测试中 → 生产 → 废弃
         ↓
       失败
```

- **草稿**：模型已创建，未进行测试
- **测试中**：模型正在测试
- **生产**：模型已发布到生产环境
- **失败**：模型测试失败
- **废弃**：模型已废弃，不再使用

**关联关系**：
- `1:N → ModelVersion`：一个模型可有多个版本
- `1:N → FineTuningJob`：模型可作为基础模型被调优
- `1:N → ModelEngineeringJob`：模型可被工程化处理
- `1:N → TrainingJob`：模型可通过训练任务生成
- `1:N → Inference`：模型可用于推理（跨域关联）

**操作**：
- `createVersion()`：创建新版本
- `publishVersion(versionId)`：发布版本
- `archiveVersion(versionId)`：归档版本
- `changeStatus(newStatus)`：状态转换

## 聚合根3：模型工程化任务（ModelEngineeringJob Aggregate）

**领域对象**：`ModelEngineeringJob`  
**标识**：`jobId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `jobId` | UUID | 任务唯一标识 |
| `name` | String | 任务名称 |
| `description` | String | 任务描述 |
| `sourceModelId` | UUID | 源模型ID |
| `optimizationTypes` | List<String> | 优化类型列表：QUANTIZATION / PRUNING / DISTILLATION / SPARSITY |
| `config` | Object | 优化配置（量化bit数、蒸馏温度等） |
| `targetHardware` | Enum | 目标硬件：CPU / GPU / NPU / EDGE_DEVICE |
| `outputFormat` | Enum | 输出格式：ONNX / TENSORRT / TFLITE / NCNN / OPENVINO |
| `outputModelId` | UUID | 输出模型ID |
| `status` | Enum | 状态（见下方状态机） |
| `conversionMetrics` | Object | 转换前后对比指标 |
| `createdBy` | UUID | 创建者ID |
| `tenantId` | UUID | 所属租户ID |
| `createdAt` | Timestamp | 创建时间 |
| `completedAt` | Timestamp | 完成时间 |

**状态机**：
```
待提交 → 等待调度 → 运行中 → 已完成
                      ↓
                    失败
```

- **待提交**：任务已创建，未提交
- **等待调度**：等待资源调度
- **运行中**：正在进行工程化处理
- **已完成**：处理完成，输出模型已生成
- **失败**：处理失败

**关联关系**：
- `N:1 → Model`：源模型
- `1:1 → Model`：输出模型

**操作**：
- `submitJob()`：提交工程化任务
- `cancel()`：取消任务
- `compareMetrics()`：对比优化前后指标

## 聚合根4：开发任务（DevelopmentTask Aggregate）

**领域对象**：`DevelopmentTask`  
**标识**：`taskId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `taskId` | UUID | 任务唯一标识 |
| `name` | String | 任务名称 |
| `description` | String | 任务描述 |
| `image` | String | 容器镜像URI（如tensorflow:latest） |
| `resourceConfig` | ResourceConfig | 资源配置 |
| `mountedDatasets` | List<UUID> | 挂载的数据集ID列表 |
| `mountedModels` | List<UUID> | 挂载的模型ID列表 |
| `status` | Enum | 状态（见下方状态机） |
| `accessType` | List<Enum> | 访问方式：JUPYTER / VSCODE / SSH |
| `accessEndpoints` | Object | 访问端点信息（URL、端口等） |
| `owner` | UUID | 所有者ID |
| `tenantId` | UUID | 所属租户ID |
| `createdAt` | Timestamp | 创建时间 |
| `startedAt` | Timestamp | 启动时间 |
| `stoppedAt` | Timestamp | 停止时间 |

**状态机**：
```
创建 → 启动中 → 运行中 → 关机 → 删除
         ↓
       启动失败
```

- **创建**：任务已创建，未启动
- **启动中**：环境正在启动
- **运行中**：环境已启动，可进行开发
- **启动失败**：启动失败
- **关机**：环境已关闭
- **删除**：环境已删除

**关联关系**：
- `N:1 → Dataset`：挂载的数据集
- `N:1 → Model`：挂载的模型
- `N:1 → ContainerImage`：使用的容器镜像

**操作**：
- `start()`：启动开发环境
- `stop()`：停止开发环境
- `restart()`：重启环境
- `mountDataset(datasetId)`：挂载数据集
- `mountModel(modelId)`：挂载模型
- `getAccessEndpoints()`：获取访问地址

## 聚合根5：训练任务（TrainingJob Aggregate）

**领域对象**：`TrainingJob`  
**标识**：`jobId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `jobId` | UUID | 任务唯一标识 |
| `name` | String | 任务名称 |
| `description` | String | 任务描述 |
| `mode` | Enum | 训练模式：SINGLE_MACHINE / DISTRIBUTED |
| `workerCount` | Integer | 分布式训练节点数（`mode=DISTRIBUTED` 时必填，最小值 2，见规则 MDL-010） |
| `image` | String | 容器镜像URI |
| `envVars` | Map<String, String> | 环境变量 |
| `command` | String | 执行命令 |
| `resourceConfig` | ResourceConfig | 资源配置 |
| `mountedDatasets` | List<UUID> | 挂载的数据集ID列表 |
| `status` | Enum | 状态（见下方状态机） |
| `tensorboardUrl` | String | TensorBoard访问地址（可选） |
| `logs` | String | 日志存储位置 |
| `outputPath` | String | 输出路径（保存模型等） |
| `createdBy` | UUID | 创建者ID |
| `tenantId` | UUID | 所属租户ID |
| `createdAt` | Timestamp | 创建时间 |
| `startedAt` | Timestamp | 开始时间 |
| `completedAt` | Timestamp | 完成时间 |

**状态机**：
```
待提交 → 等待调度 → 运行中 → 已完成
           ↓        ↓
         失败   已取消
```

- **待提交**：任务已创建，未提交
- **等待调度**：等待资源调度
- **运行中**：正在训练
- **已完成**：训练完成
- **失败**：训练失败
- **已取消**：用户取消任务

**关联关系**：
- `N:1 → Dataset`：挂载的数据集
- `1:1 → Model`：训练输出的模型（通过outputPath）
- `N:1 → ContainerImage`：使用的容器镜像

**操作**：
- `submitJob()`：提交训练任务
- `cancel()`：取消任务
- `getLogs()`：获取日志
- `getTensorboard()`：获取TensorBoard链接
