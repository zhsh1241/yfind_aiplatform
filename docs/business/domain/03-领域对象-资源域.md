# 03-领域对象 – 资源域

> **范围**：资源域负责计算集群、资源池、AI芯片、存储池和容器镜像的管理与调度。  
> **组织方式**：按聚合根分组。每个聚合根包含标识、核心属性、状态机、关联关系和子对象。

---

## 聚合根1：集群（Cluster Aggregate）

**领域对象**：`Cluster`  
**标识**：`clusterId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `clusterId` | UUID | 集群唯一标识 |
| `name` | String | 集群名称 |
| `description` | String | 集群描述 |
| `clusterType` | Enum | 集群类型：KUBERNETES / SLURM / YARN / CUSTOM |
| `status` | Enum | 集群状态：HEALTHY / DEGRADED / OFFLINE |
| `nodes` | List<ClusterNode> | 集群节点列表 |
| `totalCPU` | String | 总CPU核数（如"128"） |
| `totalGPU` | String | 总GPU数（如"16×A100"） |
| `totalNPU` | String | 总NPU数（可选） |
| `totalMemory` | String | 总内存（如"512Gi"） |
| `allocatedCPU` | String | 已分配CPU核数 |
| `allocatedGPU` | String | 已分配GPU数 |
| `allocatedMemory` | String | 已分配内存 |
| `schedulingPolicy` | Enum | 调度策略：FIFO / PRIORITY_QUEUE / FAIR_SHARE |
| `location` | String | 集群物理位置 |
| `tenantId` | UUID | 所属租户ID（可选，用于多租户场景） |
| `createdAt` | Timestamp | 创建时间 |
| `updatedAt` | Timestamp | 更新时间 |

**子对象**：`ClusterNode`

| 属性 | 类型 | 说明 |
|------|------|------|
| `nodeId` | UUID | 节点ID |
| `hostname` | String | 节点主机名 |
| `ipAddress` | String | 节点IP地址 |
| `cpuSpec` | String | CPU规格（如"64 cores Intel Xeon"） |
| `gpuSpec` | List<String> | GPU规格列表（如["A100", "A100"]） |
| `npuSpec` | List<String> | NPU规格列表（可选） |
| `memorySpec` | String | 内存规格（如"256Gi"） |
| `status` | Enum | 节点状态：HEALTHY / DEGRADED / OFFLINE / MAINTENANCE |
| `utilizationCPU` | Double | CPU利用率（0-1） |
| `utilizationMemory` | Double | 内存利用率（0-1） |
| `utilizationGPU` | Double | GPU利用率（0-1） |

**状态机**：
```
创建 → 就绪 → 维护 → 就绪
         ↓
       故障
```

- **创建**：集群已创建，正在初始化
- **就绪**：集群正常运行
- **维护**：集群进入维护模式
- **故障**：集群出现故障

**关联关系**：
- `1:N → ResourcePool`：集群可分配给多个资源池
- `1:N → AIChip`：集群包含多个AI芯片

**操作**：
- `addNode(nodeSpec)`：添加节点
- `removeNode(nodeId)`：移除节点
- `getResourceUsage()`：获取资源使用情况
- `changeStatus(newStatus)`：转换状态

## 聚合根2：资源池（ResourcePool Aggregate）

**领域对象**：`ResourcePool`  
**标识**：`poolId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `poolId` | UUID | 资源池唯一标识 |
| `name` | String | 资源池名称 |
| `description` | String | 资源池描述 |
| `clusterId` | UUID | 所属集群ID |
| `isolationMode` | Enum | 隔离模式：EXCLUSIVE（独占） / SHARED（共享） |
| `allocationMode` | Enum | 分配模式：NODE_ALLOCATION / QUOTA_ALLOCATION |
| `quotas` | ResourceQuota | 资源配额 |
| `nodeIds` | List<UUID> | 关联的节点ID列表（Node Allocation模式） |
| `assignedTenants` | List<UUID> | 分配给的租户ID列表 |
| `usageMetrics` | UsageMetrics | 使用情况统计 |
| `priority` | Integer | 资源池优先级（1-10，数字越大优先级越高） |
| `status` | Enum | 状态：ACTIVE / INACTIVE / MAINTENANCE |
| `createdAt` | Timestamp | 创建时间 |
| `updatedAt` | Timestamp | 更新时间 |

**子对象**：`ResourceQuota`

| 属性 | 类型 | 说明 |
|------|------|------|
| `cpu` | String | CPU配额（如"100" 表示100核） |
| `gpu` | String | GPU配额（如"10×A100"） |
| `npu` | String | NPU配额（可选） |
| `memory` | String | 内存配额（如"1Ti"） |
| `storage` | String | 存储配额 |
| `networkBandwidth` | String | 网络带宽配额 |

**子对象**：`UsageMetrics`

| 属性 | 类型 | 说明 |
|------|------|------|
| `usedCPU` | String | 已用CPU |
| `usedGPU` | String | 已用GPU |
| `usedMemory` | String | 已用内存 |
| `usedStorage` | String | 已用存储 |
| `cpuUtilization` | Double | CPU利用率（0-1） |
| `gpuUtilization` | Double | GPU利用率（0-1） |
| `memoryUtilization` | Double | 内存利用率（0-1） |
| `timestamp` | Timestamp | 采样时间 |

**关联关系**：
- `N:1 → Cluster`：资源池属于某个集群
- `M:N → Tenant`：资源池分配给多个租户

**操作**：
- `allocateToTenant(tenantId, quotas)`：分配资源给租户
- `reclaimFromTenant(tenantId)`：回收租户的资源
- `updateQuota(newQuota)`：更新资源配额
- `getUsageMetrics()`：获取使用指标

## 聚合根3：AI芯片（AIChip Aggregate）

**领域对象**：`AIChip`  
**标识**：`chipId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `chipId` | UUID | 芯片唯一标识 |
| `type` | Enum | 芯片类型：GPU / NPU / TPU / CPU（见下方硬件范围说明） |
| `vendor` | String | 芯片厂商（如NVIDIA、AMD、华为等） |
| `model` | String | 芯片型号（如A100、H100、V100） |
| `displayName` | String | 显示名称（如"NVIDIA A100 80GB"） |
| `memorySize` | String | 显存大小（如"80Gi"） |
| `computeCapability` | String | 计算能力（如CUDA Compute Capability） |
| `nodeid` | UUID | 所在节点ID |
| `status` | Enum | 状态：HEALTHY / DEGRADED / OFFLINE |
| `utilizationRate` | Double | 使用率（0-1） |
| `schedulingStrategy` | Enum | 调度策略：PACK / SPREAD / BEST_FIT |
| `constraints` | Object | 调度约束条件 |
| `createdAt` | Timestamp | 创建时间 |
| `updatedAt` | Timestamp | 更新时间 |

**状态机**：
```
检测 → 就绪 → 维护 → 就绪
         ↓
       故障
```

- **检测**：芯片已检测，正在初始化
- **就绪**：芯片正常可用
- **维护**：芯片进入维护状态
- **故障**：芯片出现故障

**关联关系**：
- `N:1 → ClusterNode`：芯片安装在某个节点上

**操作**：
- `updateUtilization(rate)`：更新使用率
- `changeStatus(newStatus)`：转换状态
- `getSchedulingStrategy()`：获取调度策略

> **硬件范围说明（Q-008）**：当前交付范围支持 **GPU**（NVIDIA/AMD）和 **NPU**（华为昇腾等）两类加速芯片；`type` 枚举中的 `TPU` 和 `CPU` 为预留扩展值，暂不进行资源调度。**FPGA** 已在平台术语表（`glossary.md` 缩写表）中注册为已知硬件类型，但 V1.0 版本不在当前交付范围内，不支持 FPGA 的资源注册、调度和监控；如需支持 FPGA，需在 `type` 枚举新增 `FPGA` 值并补充调度驱动适配。

## 聚合根4：存储池（StoragePool Aggregate）

**领域对象**：`StoragePool`  
**标识**：`poolId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `poolId` | UUID | 存储池唯一标识 |
| `name` | String | 存储池名称 |
| `description` | String | 存储池描述 |
| `backend` | Enum | 后端存储类型：NAS / OBJECT_STORAGE（S3） / BLOCK_STORAGE / LOCAL_DISK / DISTRIBUTED_FS（HDFS） |
| `totalCapacity` | String | 总容量（如"1000Ti"） |
| `usedCapacity` | String | 已用容量 |
| `tier` | Enum | 存储层级：HOT（热存储，常访问） / COLD（冷存储，归档） |
| `accessMode` | Enum | 访问模式：READ_WRITE / READ_ONLY |
| `replicationFactor` | Integer | 副本数（用于分布式存储） |
| `status` | Enum | 状态：ACTIVE / INACTIVE / DEGRADED |
| `backendConfig` | Object | 后端配置（连接参数等） |
| `tenantStorageAllocations` | List<StorageAllocation> | 租户存储分配列表 |
| `createdAt` | Timestamp | 创建时间 |
| `updatedAt` | Timestamp | 更新时间 |

**子对象**：`StorageAllocation`

| 属性 | 类型 | 说明 |
|------|------|------|
| `allocationId` | UUID | 分配ID |
| `tenantId` | UUID | 租户ID |
| `allocatedSize` | String | 分配大小 |
| `usedSize` | String | 实际使用大小 |
| `createdAt` | Timestamp | 创建时间 |
| `updatedAt` | Timestamp | 更新时间 |

**状态机**：
```
创建 → 就绪 → 维护 → 就绪
         ↓
       故障
```

- **创建**：存储池已创建，正在初始化
- **就绪**：存储池正常可用
- **维护**：存储池进入维护状态
- **故障**：存储池出现故障

**关联关系**：
- `M:N → Tenant`：通过StorageAllocation关联

**操作**：
- `allocateToTenant(tenantId, size)`：分配存储给租户
- `reclaimFromTenant(tenantId)`：回收租户存储
- `migrateData(sourcePool, targetPool)`：数据迁移
- `changeStatus(newStatus)`：转换状态

## 聚合根5：容器镜像（ContainerImage Aggregate）

**领域对象**：`ContainerImage`  
**标识**：`imageId: UUID`  

**核心属性**：

| 属性 | 类型 | 说明 |
|------|------|------|
| `imageId` | UUID | 镜像唯一标识 |
| `name` | String | 镜像名称 |
| `description` | String | 镜像描述 |
| `arch` | List<Enum> | 支持架构：ARM / X86 / GPU_ARM / GPU_X86 |
| `framework` | String | 框架名称（如PyTorch, TensorFlow, MXNet等） |
| `frameworkVersion` | String | 框架版本 |
| `purpose` | Enum | 用途：DEVELOPMENT / TRAINING / INFERENCE / TOOLS |
| `registryUrl` | String | 镜像仓库地址（如harbor.example.com/smp/image） |
| `baseImage` | String | 基础镜像（如ubuntu:20.04） |
| `versions` | List<ImageVersion> | 镜像版本列表 |
| `status` | Enum | 状态：DRAFT / RELEASED / ARCHIVED |
| `owner` | UUID | 所有者ID |
| `tenantId` | UUID | 所属租户ID（平台镜像可为null） |
| `createdAt` | Timestamp | 创建时间 |
| `updatedAt` | Timestamp | 更新时间 |

**子对象**：`ImageVersion`

| 属性 | 类型 | 说明 |
|------|------|------|
| `versionId` | UUID | 版本ID |
| `versionTag` | String | 版本标签（如v1.0, latest） |
| `digest` | String | 镜像摘要（SHA256） |
| `size` | Long | 镜像大小（字节） |
| `baseImageVersion` | String | 基础镜像版本 |
| `dependencies` | List<String> | 依赖列表（如CUDA 11.8, cuDNN 8.6） |
| `status` | Enum | 版本状态：DRAFT / RELEASED / ARCHIVED |
| `securityScan` | Object | 安全扫描结果 |
| `createdAt` | Timestamp | 创建时间 |
| `createdBy` | UUID | 创建者ID |

**状态机**：
```
草稿 → 发布 → 归档
         ↓
       下架
```

- **草稿**：镜像已创建，未发布
- **发布**：镜像已发布，可被使用
- **归档**：镜像已归档
- **下架**：镜像已下架，不可使用

**关联关系**：
- `1:N → ImageVersion`：一个镜像可有多个版本
- `1:N → DevelopmentTask`：镜像可用于开发环境
- `1:N → TrainingJob`：镜像可用于训练任务

**操作**：
- `createVersion(versionTag)`：创建新版本
- `publishVersion(versionId)`：发布版本
- `archiveVersion(versionId)`：归档版本
- `scanSecurity()`：执行安全扫描
- `updateDependencies(deps)`：更新依赖

## 跨聚合根关系总结

| 源聚合根 | 关联 | 目标聚合根 | 说明 |
|---------|------|-----------|------|
| Cluster | 1:N | ClusterNode | 集群包含节点 |
| Cluster | 1:N | ResourcePool | 集群分配给资源池 |
| Cluster | 1:N | AIChip | 集群包含芯片 |
| ResourcePool | N:1 | Cluster | 资源池属于集群 |
| ResourcePool | M:N | Tenant | 资源池分配给租户 |
| AIChip | N:1 | ClusterNode | 芯片安装在节点上 |
| StoragePool | M:N | Tenant | 存储分配给租户 |
| ContainerImage | 1:N | ImageVersion | 镜像有多个版本 |
| ContainerImage | 1:N | DevelopmentTask | 镜像用于开发 |
| ContainerImage | 1:N | TrainingJob | 镜像用于训练 |

```
        ↓

[DevelopmentTask]
        │
        ↓

        ↓

```
